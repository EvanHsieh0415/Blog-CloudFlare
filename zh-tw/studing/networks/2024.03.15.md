## 2.2.2 雙絞線

- `TP` ==雙絞線== _Twisted pair_

  2. `STP` ==有遮蔽雙絞線== Shielded Twisted Pair

| 名稱   | 種類   | 速度        |
| ------ | ------ | ----------- |
| Cat 3  | 電話線 | 12 Mbps     |

| Cat 5e |        | 100 Mbps    |
| Cat 6  |        | 1 Gbps      |
| Cat 7  |        | 10 Gbps     |

- `VCC`：正極
- `GND`：負極
- `STB` 機上盒 _Set Top Box_
